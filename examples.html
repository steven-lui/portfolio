SQL - Movie Database Exercise

The exercises are taken from w3resource where we want certain records from the database.

I have made a query based off of exercise 4. From the following tables write an SQL query:

  • To find movies whose reviewer is known.
  • Where release date is known
  • Return
    • Movie title
    • Year
    • Release date
    • Director's name
    • Actor's name
  • Format the output
  • Order by a value (director's last name in this case)

My query gets the required data from tables that I have use JOIN clauses on, which lets me attach multiple pieces of data together by a movie's id. I also use subqueries in certain areas, for example the first join. In this case, I wanted the movies which have a specific release date, as some of them lack it. I wanted to use the WITH clause, however the online editor did not accept such queries from my testing.

It is important to note that by using a WITH clause would make the query much more readable. Having easy to read code is important because quite often multiple people will end up using and maintaining your code. With all the code you may write, its important that you can understand it quickly in case you forget!
